## Cara Install Project
<ol>
  <li>Clone Project</li>
  <li>masuk ke Projectnya</li>
  <li>composer install</li>
  <li>npm install</li>
  <li>npm run dev</li>
  <li>cp .env.example .env</li>
  <li>php artisan migrate:fresh --seed</li>
  <li>php artisan serve</li>
  <li>buka cmd baru npm run watch</li>
</ol>
